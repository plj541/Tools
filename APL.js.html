This APL began with the work of Nikolay G. Nikolov. I provided some function models and descriptions of APL behaviour. The source and development conversations are available at:
   NGN/APL

Download APL.js.zip and unzip it locally or on a web site to get the additional APL primitives. It also provides an execution shell and workspace editor by starting either:
   index.html or APL.js.html
Run.html together with the workspace Html that users can include with:
   #include Html
provides for full screen applications.

Download APL.js.User.zip and unzip it at the same directory level as APL.js to provide additional workspaces.

The command:
   )fetch
lists additional files you can make available. The command:
   )fetch Saved Config
will update you config to recognize sample and test workspaces.
